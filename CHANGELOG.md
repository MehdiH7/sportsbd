## Changelog

### 0.1.0

- Initial release of `sportsbd`.
- Core Python API for:
  - Loading 3D CNN checkpoints (`r2plus1d_18`).
  - Running shot boundary detection on clips and full videos.
  - Evaluating against SoccerNet-style annotations.
  - Sweeping confidence thresholds.
- Command-line interface with `infer`, `eval`, and `sweep` subcommands.
